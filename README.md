# API RESTful - Avaliação Spring Boot

Esta API foi desenvolvida para gerenciar o fluxo de dados de uma aplicação de backend de forma eficiente. O sistema utiliza **Spring Data JPA** para persistência em banco de dados **MySQL** e conta com documentação interativa via **Swagger**, permitindo a gestão simplificada de recursos através de endpoints REST padronizados.

## 🛠️ Tecnologias Utilizadas (Stack)

* **Linguagem:** Java 25
* **Framework:** Spring Boot 4.0.3
* **Persistência:** Spring Data JPA
* **Banco de Dados:** MySQL
* **Documentação:** SpringDoc OpenAPI (Swagger) 3.0.1
* **Gerenciador de Build:** Maven com Wrapper (mvnw)

---

## 🚀 Como Executar
1. Configure o banco de dados MySQL no arquivo `application.properties`.
2. No terminal, use o comando:
   - Windows: `mvnw.cmd spring-boot:run`
   - Linux/Mac: `./mvnw spring-boot:run`

---

## 📖 Documentação da API
Após rodar o projeto, acesse o menu interativo de endpoints em:
